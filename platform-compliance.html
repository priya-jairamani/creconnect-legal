2. Compliance with Meta Platform Terms

CreConnect confirms compliance with Meta's Platform Terms and Developer Policies. Specifically:

  • We request only the permissions necessary for our app's core functionality (Principle of Minimal Permissions).
  • We provide clear and accurate descriptions for all requested permissions during the Facebook Login flow.
  • We maintain a publicly accessible Privacy Policy that accurately describes our data practices.
  • We maintain a publicly accessible Terms of Service that governs user obligations.
  • We maintain a Data Deletion Instructions page and honor user deletion requests within 30 days.
  • We do not sell, transfer, or license Meta Platform Data to third parties.
  • We do not use Meta Platform Data to build advertising audiences or for data broker activities.
  • We do not use Meta Platform Data for any purpose not disclosed to users and not approved by Meta.
  • We implement appropriate technical and organizational security measures to protect Meta Platform Data.
  • We provide users the ability to disconnect their Facebook/Instagram account at any time.
  • We comply with Meta's Community Standards and do not facilitate content that violates them.

4. Facebook Login Compliance

CreConnect's implementation of Facebook Login adheres to all Meta requirements:

  • Facebook Login is presented as one of multiple login options, not as the sole method.
  • Users are informed about what data will be accessed before authorizing Facebook Login.
  • The Facebook Login dialog accurately reflects the permissions requested.
  • We do not request permissions beyond what is listed in our approved permission set.
  • Users can complete all core account actions (login, logout, account management) without Facebook Login.
  • Disconnecting Facebook Login does not delete the user's CreConnect account unless they separately request account deletion.
  • We use only official Facebook SDK methods for authentication; we do not scrape or bypass the login flow.

5. Instagram Graph API Compliance

CreConnect's use of the Instagram Graph API complies with Meta's API policies:

  • We access only Instagram Business Accounts and Creator Accounts, not personal accounts, via the Graph API.
  • We access media, insights, and account information only for authenticated users who have explicitly granted permission.
  • Instagram data is displayed only to the account owner and, where explicitly authorized, to connected brand partners for campaign verification.
  • We do not scrape, harvest, or store Instagram data beyond what is returned via authorized API calls.
  • We implement proper error handling and respect API rate limits.
  • We refresh access tokens appropriately and do not store long-lived tokens beyond their expiration period.
  • Stored Instagram data is deleted upon user account deletion or upon revocation of Instagram permissions.

6. Data Handling and Security Compliance

6.1 Data Protection Measures

  • All data in transit is encrypted using TLS 1.2+ (HTTPS).
  • Passwords are stored as bcrypt hashes; plaintext passwords are never stored or logged.
  • Access tokens are short-lived (15-minute JWT access tokens, 7-day refresh tokens).
  • Role-based access control (RBAC) ensures users can only access data appropriate to their role (Creator, Brand, Admin).
  • File uploads are stored securely via Cloudinary with access restrictions.
  • System activity is logged via Winston for security monitoring and audit purposes.
  • Redis caching is used with appropriate TTL controls to prevent stale session data.

6.2 Meta Platform Data Restrictions

CreConnect strictly enforces the following restrictions on Meta Platform Data:

  • We do NOT sell Meta Platform Data to any third party.
  • We do NOT use Meta Platform Data to target users with advertising.
  • We do NOT transfer Meta Platform Data to data brokers or aggregators.
  • We do NOT use Meta Platform Data for profiling purposes unrelated to the Service.
  • We do NOT use Meta Platform Data for purposes not disclosed to users.
  • We do NOT store Meta Platform Data longer than necessary for the disclosed purpose.
  • We do NOT share Meta Platform Data with unauthorized third parties.

7. User Rights and Control

CreConnect ensures users maintain full control over their data and social media connections:

  • Users can view all connected social media accounts in their account settings.
  • Users can disconnect Facebook/Instagram at any time via CreConnect settings.
  • Users can delete their entire account and all associated data via account settings.
  • Users can submit a formal data deletion request via our Data Deletion Request page.
  • We honor all user rights requests (access, correction, deletion, portability) within 30 days.
  • Users are notified via email upon completion of their data deletion request.

9. Incident Response

In the event of a data security incident involving Meta Platform Data, CreConnect commits to:

  • Notifying Meta at developers.facebook.com within 72 hours of becoming aware of a breach involving Meta Platform Data.
  • Notifying affected users within the timeframe required by applicable law.
  • Taking immediate steps to contain, investigate, and remediate the incident.
  • Cooperating fully with Meta's investigation of any reported security incident.
