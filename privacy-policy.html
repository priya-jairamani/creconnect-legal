9. Data Security

We implement industry-standard technical and organizational measures to protect your personal data against unauthorized access, loss, destruction, alteration, or disclosure, including:

  • Encryption: Data in transit is protected using TLS/HTTPS. Passwords are hashed using industry-standard cryptographic algorithms (bcrypt).
  • Access Controls: Role-based access control (RBAC) with least-privilege principles. JWT-based authentication with short-lived access tokens (15 minutes) and refresh tokens (7 days).
  • Authentication: Email verification with OTP (One-Time Passwords) for account actions.
  • Monitoring: System logs monitored for suspicious activity using Winston logging.
  • Third-Party Security: Media files stored securely via Cloudinary with access control.

While we take reasonable steps to protect your data, no security system is impenetrable. In the event of a data breach that affects your personal data, we will notify you in accordance with applicable law.
